# YouTube Transcript Extension

The **YouTube Transcript Extension** lets you extract and summarize the transcript of any YouTube video directly from your browser — with a clean interface, AI-powered summaries, and options to copy or download the content.

---

## ✨ What's New

- ✅ Smart Transcript: Automatically fetches and **caches** the video transcript in Firebase.
- ✅ Smart Summary: Generates an AI summary using OpenAI and stores it for future access.
- ✅ Toggle between **📜 Transcript** and **🧠 Summary** with one click.
- ✅ Copy or Download the selected content (transcript or summary).
- ✅ Refresh button to re-fetch and regenerate both.
- ✅ Minimalist design and fully self-contained — no need to visit external sites.

---

## 📦 How It Works

This extension communicates with two backend services:

1. **Transcript Service**  
   - `POST /smart-transcript`  
   - Fetches and caches the transcript using the video URL.
   - Stores the transcript in Firestore for future access.

2. **Summary Service**  
   - `POST /smart-summary`  
   - Uses OpenAI to summarize the transcript (sent from the extension).
   - Caches the summary in Firestore as well.

These services should already be set up and deployed (e.g. on your own server), and the extension simply calls them.

---

## 💡 Features in Action

| Feature | Description |
|--------|-------------|
| 📜 **Transcript** | Auto-fetched from YouTube and stored in Firebase |
| 🧠 **Summary** | Generated by OpenAI when clicked — saved to Firebase |
| 📋 **Copy** | Copies current content (based on toggle) |
| 💾 **Download** | Saves `.txt` file named like `videoId_transcript.txt` |
| 🔁 **Refresh** | Clears cache and re-fetches both transcript and summary |

## 🚀 Install Locally

1. Clone the repo:
   ```bash
   git clone https://github.com/andresz74/youtube-transcript-extension.git
   ```
2. Open Chrome → go to `chrome://extensions/`
3. Enable **Developer Mode**
4. Click **Load Unpacked** and select the project folder

That’s it!

---

## ⚙️ Backend Requirement

This extension requires the following backend services (hosted by you):

- [`youtube-transcript-generator`](https://github.com/andresz74/youtube-transcript-generator)  
- [`chat-gpt-access`](https://github.com/andresz74/ChatGptAccess)

Make sure to update the correct URLs in `popup.js` under:

```js
const SMART_TRANSCRIPT_ENDPOINT = 'https://your-service.com/smart-transcript';
const SMART_SUMMARY_ENDPOINT = 'https://your-service.com/smart-summary';
```
